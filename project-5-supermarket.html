Supermarket Sales & Growth Analysis

End-to-end Retail Analytics using Python (Pandas) and Power BI.

Business Context & The Challenge

This project analyzes historical sales data to uncover trends in product popularity and customer profitability. A significant challenge was discovered during the initial audit: Data Integrity Issues. Many products were misclassified (e.g., 'Apple' labeled as 'Beverages'), requiring extensive cleaning before analysis could begin.

Key Findings & Value Delivered

1. Data Quality Audit

Finding: Identified significant inconsistencies in product categorization (over 70% error rate in specific samples).

Value: Created a cleaning script using Pandas to re-map products to their correct categories, ensuring 100% accuracy for the final reporting.

 

2. Revenue Drivers

Finding: After cleaning, Personal Care emerged as the top-performing category ($27k revenue), followed closely by Stationery.

Value: This insight corrects previous assumptions that were based on faulty data, allowing inventory managers to stock the correct high-demand items.

Technical Implementation: Python Code

Below is the actual Python code I wrote to clean the dataset and perform the exploratory data analysis (EDA). Note the explicit data correction block:

import pandas as pd
import matplotlib.pyplot as plt

# 1. Loading Data
sales = pd.read_csv('sales.csv')

# 2. Data Cleaning & Correction (CRITICAL STEP)
# Found mismatches between Product Name and Category. 
# Creating a corrected dataset 'new_sales'.
new_sales = sales.copy()

new_sales.loc[new_sales['product_name'] == 'Apple', 'product_category'] = 'Fruits'
new_sales.loc[new_sales['product_name'] == 'Orange Juice', 'product_category'] = 'Beverages'
new_sales.loc[new_sales['product_name'] == 'Notebook', 'product_category'] = 'Stationery'
new_sales.loc[new_sales['product_name'] == 'Detergent', 'product_category'] = 'Household'
new_sales.loc[new_sales['product_name'] == 'Shampoo', 'product_category'] = 'Personal Care'

# 3. Analysis on Cleaned Data
# Analyzing Total Sales by Product Category
category_sales = new_sales.groupby('product_category')['total_price'].sum().sort_values(ascending=False)
print("Top Categories by Revenue:\n", category_sales)

# Visualizing Sales Distribution by City
sales.groupby('city')['total_price'].sum().plot(kind='pie', autopct='%1.1f%%')
plt.title("Sales by City")
plt.show()

# Identifying Most Valuable Customers
# Result: Female Members spend the most on average.
valuable_customers = sales.groupby(['gender','customer_type'])['total_price'].mean().sort_values()
