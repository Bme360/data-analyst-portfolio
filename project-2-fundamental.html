01 Scope & Objective

This project performs a full Fundamental Analysis of three major cybersecurity companies — comparing financial performance, stability, and efficiency across 5 years (20 quarters per company).

The objective: identify the strongest and weakest players based purely on quantitative indicators. SQL was used to structure all raw financial data before visualization in Power BI.

CrowdStrike (CRWD)

Cloud-native endpoint security leader — analyzed across profitability and growth metrics.

Zscaler (ZS)

Zero-trust cloud security platform — evaluated for cash generation and margin efficiency.

Palo Alto Networks (PANW)

Legacy security giant transitioning to SaaS — debt structure and operational metrics compared.

02 Data Sourcing & Preparation

Financial statement data was extracted from Income Statements, Balance Sheets, and Cash Flow Statements covering 20 quarters per company. SQL was used to normalize and structure all data, ensuring accurate and comparable cross-company results.

Report Type Purpose Key Fields
Income Statement Profitability & efficiency metrics Revenue, Gross Profit, Operating Income, Net Income
Balance Sheet Financial stability & capital structure Total Assets, Liabilities, Equity Ratios
Cash Flow Statement Liquidity & cash generation Free Cash Flow, Operating Cash Flow

03 SQL Data Modeling

A relational schema was designed to connect financial reports across time periods and companies, enabling efficient querying for trend analysis and ratio calculations.

Design Decision

By normalizing all three companies into a single relational model, queries can compare all companies simultaneously in one SQL statement — no manual copy-paste, no errors, full auditability.

Sample: Metric Calculation Query SQL 
/* 
  Calculate core financial ratios for all companies per quarter.
  Joining Income Statement + Balance Sheet + Cash Flow data.
*/

SELECT 
    c.company_name,
    f.fiscal_quarter,
    f.fiscal_year,
    
    -- Profitability Ratios
    ROUND((f.gross_profit / NULLIF(f.revenue, 0)) * 100, 2)        AS gross_margin_pct,
    ROUND((f.operating_income / NULLIF(f.revenue, 0)) * 100, 2)    AS operating_margin_pct,
    ROUND((f.net_income / NULLIF(f.revenue, 0)) * 100, 2)          AS net_margin_pct,
    
    -- Cash Generation
    ROUND((cf.free_cash_flow / NULLIF(f.revenue, 0)) * 100, 2)     AS fcf_margin_pct,
    
    -- Capital Structure
    ROUND(bs.total_debt / NULLIF(bs.shareholders_equity, 0), 2)    AS debt_to_equity,
    
    -- Growth (QoQ)
    ROUND(((f.revenue - LAG(f.revenue) OVER 
        (PARTITION BY c.company_id ORDER BY f.fiscal_year, f.fiscal_quarter))
        / NULLIF(LAG(f.revenue) OVER 
        (PARTITION BY c.company_id ORDER BY f.fiscal_year, f.fiscal_quarter), 0)) * 100, 2) 
        AS revenue_growth_qoq

FROM financials f
JOIN companies c        ON f.company_id = c.company_id
JOIN balance_sheet bs   ON f.company_id = bs.company_id AND f.period_id = bs.period_id
JOIN cash_flows cf      ON f.company_id = cf.company_id AND f.period_id = cf.period_id
ORDER BY c.company_name, f.fiscal_year, f.fiscal_quarter;

04 Core Metrics Calculated

Revenue Growth Rate (QoQ & YoY) Gross Margin Operating Margin Net Margin Free Cash Flow Margin Debt-to-Equity Ratio

All metrics were calculated directly in SQL using Window Functions, CTEs, and JOIN logic, then exported to Power BI for visualization.
