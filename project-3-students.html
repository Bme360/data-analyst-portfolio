03 SQL Query Example

Sample query used for data preparation — counting students where Math is their peak score, filtered to a specific ID range:

Peak Score Logic — ID Range Filter SQL 
/*
  Look at IDs 40 through 50 (inclusive).
  Count students whose highest score among the three subjects
  (Math, Reading, Writing) is MathScore — and is strictly
  higher than both other scores.
*/

SELECT COUNT(StudentID) AS StudentCount
FROM (
    SELECT 
        s.StudentID,
        s.mathScore,
        s.writingScore,
        s.readingScore
    FROM student s
    WHERE s.mathScore > s.readingScore 
      AND s.mathScore > s.writingScore
) ranked
WHERE StudentID BETWEEN 40 AND 50;

Why this matters

By identifying each student's dominant subject area, the analysis can segment learners into profiles — helping educators tailor support to individual learning strengths rather than treating all students uniformly.
