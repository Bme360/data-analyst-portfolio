Smart Road Safety

Interactive Dashboard (Power BI)

The Challenge

In this project, I analyzed data from an intelligent transportation alert system in the US. The goal was to identify 'blind spots' where the system fails to alert to accidents, and to find patterns of serious accidents in different countries.

The Tech Stack

DAX (Business Intelligence): I created advanced metrics in Power BI to calculate the severity rate of accidents and the reliability rating of the system.

Data Visualization: I built an interactive dashboard that allows decision makers to identify in seconds where the system requires improvement.

Project Overview & Data Preparation

This project focuses on auditing the reliability and efficiency of a smart transportation alert system to identify critical "blind spots" and factors contributing to road accidents. The dataset, comprising approximately 4756 traffic records, was subjected to a comprehensive ETL (Extract, Transform, Load) process to ensure data integrity and analytical readiness.

The analyzed variables included: accident severity (Fatal, Major, Minor), weather conditions, road IDs, geographical locations (USA States), and system-generated alerts. By architecting a relational database in SQL, I performed rigorous data cleaning and feature engineering. This enabled the calculation of advanced metrics, such as "False Negative" alert rates and "Accident Streaks" using complex window functions and Gaps-and-Islands logic.

SQL Query Example

Which combination (road and weather) has the highest failure rate?:


/*
Which combination (road and weather) has the highest failure rate?
*/

select s.RoadID,s.Weather
		,sum(case
				when s.AccidentOccurred='Yes' then 1 end ) as AccidentOccurred
			,sum(case
				when s.AlertGenerated='No' and s.AccidentOccurred='Yes'then 1 end )as n
		,cast(sum(case
				when s.AlertGenerated='No' and s.AccidentOccurred='Yes'then 1 end ) as float)/cast(sum(case
				when s.AccidentOccurred='Yes' then 1 end )as float)
				*100as MissRate
from SyntheticTransportation s
group by s.RoadID,s.Weather
having sum(case
				when s.AccidentOccurred='Yes' then 1 end ) >=20
