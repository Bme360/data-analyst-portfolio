Smart Road Safety: Alert System Audit

A Data-Driven Audit of Intelligent Transportation Systems using SQL & Power BI.

Project Overview & The Challenge

In this project, I acted as a Data Analyst auditing an intelligent transportation alert system in the US. The primary objective was to evaluate the system's reliability by identifying "Blind Spots"â€”scenarios where accidents occurred, but the system failed to generate an alert.

The dataset comprised approximately 4,750 traffic records. I performed a comprehensive ETL process to clean the data and engineered metrics to calculate the system's "Miss Rate" (False Negatives).

Key Findings

  • Weather Impact: High correlation found between adverse weather (Fog/Rain) and increased system failure rates.
  • Blind Spots: Identified specific Road IDs with a high frequency of un-alerted accidents.
  • System Reliability: Calculated the overall failure rate to help engineers prioritize firmware updates.

Tech Stack & Skills

  • Advanced SQL: Window Functions, CTEs, and Logic for "False Negative" detection.
  • Power BI (DAX): Created measures for Severity Rate and dynamic reliability KPIs.
  • Data Modeling: Built a relational schema to connect Road, Weather, and Accident tables.

Technical Deep Dive: SQL Logic

One of the core challenges was isolating specific combinations of Road IDs and Weather conditions that had a statistically significant failure rate. The query below calculates the "Miss Rate": the percentage of accidents where no alert was generated, filtered for roads with at least 20 accidents.

/* Query: Identifying High-Risk Failure Zones
   Objective: Calculate the 'Miss Rate' (No Alert during Accident) 
   grouped by Road and Weather conditions.
*/

SELECT 
    s.RoadID,
    s.Weather,
    -- Count total accidents
    SUM(CASE WHEN s.AccidentOccurred = 'Yes' THEN 1 ELSE 0 END) AS Total_Accidents,
    
    -- Count missed alerts (False Negatives)
    SUM(CASE WHEN s.AlertGenerated = 'No' AND s.AccidentOccurred = 'Yes' THEN 1 ELSE 0 END) AS Missed_Alerts,
    
    -- Calculate Miss Rate Percentage
    CAST(SUM(CASE WHEN s.AlertGenerated = 'No' AND s.AccidentOccurred = 'Yes' THEN 1 END) AS FLOAT) 
    / 
    CAST(SUM(CASE WHEN s.AccidentOccurred = 'Yes' THEN 1 END) AS FLOAT) * 100 AS Miss_Rate_Percent

FROM SyntheticTransportation s
GROUP BY s.RoadID, s.Weather
HAVING SUM(CASE WHEN s.AccidentOccurred = 'Yes' THEN 1 END) >= 20
ORDER BY Miss_Rate_Percent DESC;

This analysis allows stakeholders to focus maintenance efforts on specific road segments rather than checking the entire network, saving operational costs and improving safety.
